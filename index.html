Projects

ModelHub — Deep Learning Model Platform

Comprehensive platform for sharing, discovering, and deploying deep learning models. Features model discovery with filtering, user authentication, model management with metadata, and deployment capabilities. Built with FastAPI backend, Next.js frontend, PostgreSQL/SQLite database, JWT authentication, and TypeScript. Includes responsive design, theme toggle, performance metrics tracking.

Semantic ArXiv — AI‑Powered Search Engine

Deep learning-powered semantic search engine for academic papers from arXiv. Built with FastAPI backend using sentence transformers and FAISS for vector similarity search, Next.js 14 frontend with TypeScript and Tailwind CSS. Features real-time semantic search by meaning, not just keywords, with modern responsive UI and Docker deployment.

DCGAN CelebA Face Generation

Deep Convolutional Generative Adversarial Network (DCGAN) for generating realistic human faces using the CelebA dataset. Implemented custom Generator and Discriminator networks with adversarial training, featuring early stopping mechanisms, automatic image generation, configurable hyperparameters, and training loss visualization. Built with TensorFlow, supports large-scale datasets with 64x64 pixel output resolution.

Human Emotion Detection

Machine learning project for detecting human emotions from images using PyTorch. Implements a convolutional neural network for emotion classification across three categories: Angry, Happy, and Sad. Features modular architecture with separate components for data loading, model training, evaluation, and visualization. Includes GPU optimization, configurable parameters, and comprehensive training pipeline for computer vision applications.

Videoparty — Watch Party Platform

Real-time watch party platform enabling users to create rooms and watch videos together without authentication. Features auto-detection of streaming platforms (YouTube, Vimeo, Twitch), playlist mode, AI-powered content recommendations, and real-time video synchronization. Initially built with Go backend, later migrated to FastAPI for enhanced performance. Frontend built with React.
